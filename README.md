# 🚀 CareerPilot AI

CareerPilot AI is an AI-powered Career Guidance Agent designed to help students analyze their skills, explore career opportunities, find internships, improve resumes, and prepare for interviews.

## 🎯 Features

- 🧠 Skill Analyzer
- 🎯 Career Recommender
- 📚 Learning Path Generator
- 💼 Internship Matcher
- 📄 Resume Analyzer
- 🎤 Interview Coach
- 🤖 AI Career Chatbot
- 📊 Career Dashboard
- 📄 PDF Career Report Generator

## 🛠 Technologies Used

- Python
- Streamlit
- Google Gemini API
- Pandas
- ReportLab

## 🚀 Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
