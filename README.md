# 🎓 AI Agent Career Guidance for Students

An AI-powered Career Guidance System that helps students make informed career decisions through personalized recommendations, resume analysis, skill assessment, internship matching, interview preparation, and AI-driven career assistance.

---

## 📌 Project Overview

Choosing the right career can be challenging for students due to the overwhelming number of career paths, courses, and skill requirements. This project provides an intelligent AI-powered platform that assists students in identifying suitable career options based on their interests, skills, education, and career goals.

The system combines Artificial Intelligence, Natural Language Processing (NLP), and data-driven recommendations to deliver personalized career guidance and learning support.

---

# 🎯 Problem Statement

Many students struggle with:
- Choosing the right career path
- Understanding required skills
- Finding relevant internships
- Preparing resumes
- Interview preparation
- Planning a structured learning roadmap

This project aims to solve these problems through an AI-powered virtual career assistant.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/PoojithaPerumalla/AI-Agent-Career-Guidance-For-Students.git
```

Move into project folder

```bash
cd AI-Agent-Career-Guidance-For-Students
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
